## MDC知識管理戦略

### MDCルール構成
```
.cursor/rules/
├── team-standards.mdc      # チーム基本標準（Always Rules）
├── [技術名]-patterns.mdc   # 技術別パターン（Auto Rules）
├── debug-workflow.mdc      # デバッグ支援（Agent Rules）
├── code-review.mdc         # レビュー支援（Manual Rules）
└── project-specific.mdc    # プロジェクト固有ルール
```

### 自動適用戦略
- **Always Rules**: 全ファイルで常時適用される基本ルール
- **Auto Rules**: ファイルタイプに応じた自動適用ルール
- **Agent Rules**: AI判断による適切なタイミングでの適用
- **Manual Rules**: 必要時の手動適用ルール

### 知見蓄積戦略
- **自動記録**: 問題解決時の自動的な知見記録
- **パターン抽出**: 繰り返し作業からのパターン自動抽出
- **継続改善**: 使用状況に基づくMDCルール自動最適化
