# プロジェクト概要・コンテキスト

## 📋 ファイル概要

**目的**: プロジェクトの基本情報、技術スタック、アーキテクチャ概要などのコンテキスト情報を記録
**使用場面**: Cursor AI が適切な提案を行うために必要な背景情報を提供
**自動参照**: `.cursor/rules/project-context.mdc` から自動参照され、プロジェクト全体の理解が必要な際に自動的に提供されます

## 🎯 主な用途

- **プロジェクト基本情報**: 目的、期間、チーム構成などの記録
- **技術スタック管理**: 使用技術の整理と制約の明記
- **アーキテクチャ概要**: システム構成とデータフローの文書化
- **開発環境情報**: ツール、フロー、ルールの記録

## 📚 詳細な使用方法

**完全な記載例とベストプラクティス**: [docs/templates/context-guide.md](../../docs/templates/context-guide.md) を参照してください。

> **💡 .cursor/rules形式での活用**: このファイルは `.cursor/rules/project-context.mdc` から自動参照されます。
> プロジェクト全体の理解が必要な際に、.cursor/rulesが自動的にこのコンテキストを提供します。
