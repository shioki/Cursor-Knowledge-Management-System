# パターンを登録

新しい実装パターンをパターンライブラリに登録します。

## 手順

1. ユーザーに以下の情報をヒアリングしてください:
   - **パターン名**: 具体的で分かりやすい名前
   - **目的**: 解決する問題、適用場面、期待効果
   - **実装例**: 具体的なコード例
   - **注意事項**: 制約、パフォーマンス影響、メンテナンス性

2. `.claude/skills/pattern-library/references/PATTERNS_TEMPLATE.md`（または `.cursor/skills/...`）を読み込み、既存のパターンを確認してください。重複がないか確認します。

3. 以下のテンプレートでエントリを作成し、適切なセクションに追記してください:

```markdown
## パターン名: [具体的な名前]

> 登録日: YYYY-MM-DD

### 目的
- **解決する問題**:
- **適用場面**:
- **期待効果**:

### 実装例
\```[言語]
// コード例
\```

### 使用上の注意
- **制約事項**:
- **パフォーマンス影響**:
- **メンテナンス性**:

### 関連パターン
- **組み合わせ可能**:
- **代替パターン**:
```

4. 登録が完了したら、既存の関連パターンとの紐付けを提案してください。

## 注意事項

- 実装例は具体的で動作するコードを含めてください
- 適用場面を明確にし、過剰な一般化を避けてください
- 類似パターンが既にある場合は統合を検討してください
