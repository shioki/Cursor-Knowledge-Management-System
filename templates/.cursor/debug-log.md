# デバッグ・問題解決ログ

## 📋 ファイル概要

**目的**: プロジェクトで発生した問題、エラー、パフォーマンス課題などの解決過程を記録
**使用場面**: 同様の問題の再発防止と、チームの問題解決能力向上に活用
**自動参照**: `.cursor/rules/debug-support.mdc` から自動参照され、類似問題発生時に自動的に過去の解決策を提案します

## 🎯 主な用途

- **問題・エラーログ**: 発生した問題の詳細記録と解決過程
- **パフォーマンス問題**: 性能課題の分析と最適化記録
- **環境固有問題**: 特定環境での問題と回避策
- **デバッグ手順**: 効率的な問題解決プロセスの文書化

## 📚 詳細な使用方法

**完全な記載例とベストプラクティス**: [docs/templates/debug-guide.md](../../docs/templates/debug-guide.md) を参照してください。

> **💡 .cursor/rules形式での活用**: このファイルは `.cursor/rules/debug-support.mdc` から自動参照されます。
> 類似問題発生時に、.cursor/rulesが自動的に過去の解決策を提案します。
